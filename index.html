What is a viewbox?

The viewBox is an attribute of the SVG element in HTML. It is used to scale the SVG element that means we can set the coordinates as well as width and height.

source: https://www.geeksforgeeks.org/svg-viewbox-attribute/

What is a viewport?

The viewport is the visible area of the SVG image. An SVG image can logically be as wide and high as you want, but only a certain part of the image can be visible at a time. The area that is visible is called the viewport.

source: http://tutorials.jenkov.com/svg/svg-viewport-view-box.html
